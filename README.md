About
This is a simple Java simulator for ISO 15939 measurement. It helps to calculate software quality in 5 steps.


How to Run
I used only standard Java library.
You can run it from terminal:  Compile: javac *.java   Run: java Main 


Steps in App
Profile: Enter user name and school.
Define: Pick a scenario.
Plan: See the metrics table.  
Collect: Enter values and see 1-5 scores.  
Analyse: Final results and gap analysis
